GPS Testing 
Some folks want to test the GPS on the Jetpack hardware.

For that audience, a special program exists to do just that.

This software IS NOT suitable for use as a tracker.

It is only meant to be used when testing the GPS portion of the tracker hardware.

This software will not enter Configuration Mode or Flight Mode.

Instead, the behavior is:

    

  • The tracker takes no action on its own, it is idle
    • 

  • The radio is defaulted to being turned off
    • 

  • The GPS is enabled, and all NMEA sentences are enabled
    • 

  • The tracker will respond to JSON API messages
    •
